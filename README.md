# Clash Rules from domain-list-community

这些规则文件由 [v2fly/domain-list-community](https://github.com/v2fly/domain-list-community) 自动转换生成。

**最后更新**: 2026-01-29 04:09:57 UTC

## 使用方法

### Classical 格式 (behavior: classical)

```yaml
rule-providers:
  google:
    type: http
    behavior: classical
    url: "https://raw.githubusercontent.com/cs3306/domain-list-clash/release/classical/google.yaml"
    path: ./ruleset/google.yaml
    interval: 86400

rules:
  - RULE-SET,google,PROXY
```

### Domain 格式 (behavior: domain)

```yaml
rule-providers:
  google:
    type: http
    behavior: domain
    url: "https://raw.githubusercontent.com/cs3306/domain-list-clash/release/domain/google.yaml"
    path: ./ruleset/google.yaml
    interval: 86400

rules:
  - RULE-SET,google,PROXY
```
